# Shashwat Vatsyayan — Cinematic Portfolio 2026

A premium, immersive digital identity for a **Creative Technologist, Filmmaker, and Student Leader**. Built with a focus on cinematic storytelling, smooth 3D scroll animations, and a modern technical aesthetic.

## 🎬 Live Experience
The website features a high-performance canvas frame-sequence engine that scrubs through cinematic footage as the user scrolls, creating a truly immersive storytelling experience.

## 🛠️ Technical Stack
- **Framework:** Next.js 16 (App Router)
- **Animation:** Framer Motion & Scroll-driven Canvas Engine
- **Styling:** Tailwind CSS 4
- **Smooth Scroll:** Lenis
- **Typography:** Geist Sans & Mono
- **Icons:** Phosphor Icons

## 🚀 Features
- **Cinematic Scroll:** Interactive video storytelling pinned to the viewport.
- **Premium UI:** Neumorphic and glassmorphic components with depth.
- **Project Showcase:** Interactive hover-effects for futuristic concept projects.
- **Education Timeline:** A visually distinguished timeline of academic milestones.
- **Tech Stack:** Categorized technical and creative toolkits.

## 📦 Project Structure
```text
src/
├── app/            # Next.js App Router & Layout
├── components/
│   ├── sections/   # Major page sections (Hero, Projects, etc.)
│   ├── ui/         # Reusable Neumorphic/Glassmorphic atoms
│   └── providers/  # Global context (Smooth Scroll)
└── public/
    └── frames/     # 120+ High-quality animation frames
```

## 🛠️ Getting Started

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run Development Server:**
   ```bash
   npm run dev
   ```

3. **Build for Production:**
   ```bash
   npm run build
   ```
